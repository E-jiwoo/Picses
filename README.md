## Project Overview

Picses은 교내 도서 대출 반납 시스템입니다.
<br/>
##  Background

점심, 저녁시간에는 밥먹고 매점가고 놀기에도 바쁘다보니 쉬는 시간에 도서관을 이용하고 싶지만, 점심과 저녁시간에만 운영되는 학교의 도서관으로 인해 학생 스스로 웹페이지를 가지고 대출 및 반납을 할 수 있는 서비스를 기획 및개발하게 되었습니다.
